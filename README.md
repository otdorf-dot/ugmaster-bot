# 🤖 Telegram бот «Юг Мастер»

AI-консультант строительной компании для Telegram.  
Отвечает на вопросы, показывает фото работ, собирает заявки.

---

## 🚀 Запуск на Railway (бесплатно)

### Шаг 1 — Получите токен бота у @BotFather

1. Откройте Telegram, найдите **@BotFather**
2. Напишите `/start`, затем `/newbot`
3. Введите имя: `Юг Мастер`
4. Введите username: `UgMaster134bot`
5. Скопируйте полученный **токен** (вида `7123456789:AAF...`)

### Шаг 2 — Получите Anthropic API ключ

1. Зайдите на [console.anthropic.com](https://console.anthropic.com)
2. Раздел **API Keys** → **Create Key**
3. Скопируйте ключ (вида `sk-ant-...`)

### Шаг 3 — Загрузите код на GitHub

1. Зарегистрируйтесь на [github.com](https://github.com)
2. Создайте новый репозиторий (New repository) → назовите `ugmaster-bot`
3. Загрузите файлы `index.js` и `package.json` в репозиторий

### Шаг 4 — Деплой на Railway

1. Зайдите на [railway.app](https://railway.app) → **Login with GitHub**
2. Нажмите **New Project** → **Deploy from GitHub repo**
3. Выберите репозиторий `ugmaster-bot`
4. Railway автоматически определит Node.js проект

### Шаг 5 — Добавьте переменные окружения

В Railway откройте ваш проект → вкладка **Variables** → добавьте:

| Переменная | Значение |
|---|---|
| `TELEGRAM_TOKEN` | токен от @BotFather |
| `ANTHROPIC_API_KEY` | ключ от console.anthropic.com |

### Шаг 6 — Запустите!

Railway автоматически запустит бота после добавления переменных.  
Зайдите в Telegram → найдите **@UgMaster134bot** → нажмите **Start** 🎉
